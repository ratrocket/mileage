techniques

to help working with non-ascii: 

    char to ascii value:
    ?a gets 97
    ?\n gets 10
    
    string to integer
    'a'[0] gets 97
    in ruby 1.9 apparently that won't work, use ord:
    'a'.ord gets 97
    
    [MOST IMPORTANTLY] to go from integer to char use chr:
    97.chr gets "a"
    10.chr gets "\n"
    
    To inspect a fucked up string, do
    
    => "AV Club Album Review “Black Up” June 2011"
    >> a.title.each_byte do |c|
    ?>   puts c
    >>   puts c.chr
    >>   puts "======================="
    >> end

decimal to hex conversion: 

    10.to_s(16) gets "a"
    255.to_s(16) gets "ff"

Arrays to hashes, etc. 

    attrs = %w[val1 val2 val3]
    ["val1", "val2", "val3"]

    values = ["one", "two", "three"]
    ["one", "two", "three"]

    attrs.zip values
    [["val1", "one"], ["val2", "two"], ["val3", "three"]]

    Hash[*(attrs.zip(values)).flatten]
    {"val1": "one", "val2": "two", "val3": "three"}
